    ds.createjs.SelectUpImager(width, height, container, opts)

    实现createjs下选择图片上传，触摸的 拖拽、旋转、缩放

    new SelectUpImager(width, height, container, opts)
    createjs/SelectUpImager.js, line 31

    createjs下选择图片上传管理器

    Parameters:
    Name Type Description
    width number

    图片显示容器宽
    height number

    图片显示容器高
    container createjs.Container

    选择好的图片会放添加到什么容器内，可以不设置
    opts object

    图片选择器的配置参数

    Properties
    Name Type Attributes Default Description
    size number

    选择器对选择的图片进行缩放尺寸。如：设置100，那选择上传的图片的图片 宽与高 都不可能小于100px。默认不设置会使用width或height的最大值
    layout string <optional>
    		 'outSide'

    默认：'outSide' 撑满容器 使用'inSide' 缩放到容器内

    • SelectUpImager.LAYOUT_OUTSIDE='outSide'
    • SelectUpImager.LAYOUT_INSIDE='inSide'
    scaleMax number <optional>
    		 0.2

    允许缩放最大比例
    scaleMin number <optional>
    		 3

    允许缩放最小比例
    look boolean <optional>
    		 false

    是否锁定 锁定不能缩放 不能拖动 不能旋转
    drag boolean <optional>
    		 true

    是否可以拖拽
    zoom boolean <optional>
    		 true

    是否可以缩放
    rotation boolean <optional>
    		 true

    是否可以旋转
    dragRect createjs.Rectangle <optional>

    设置拖拽访问
    hasRect boolean <optional>
    		 false

    设置true 按width height 创建拖拽范围
    clear boolean <optional>
    		 true

    选择新相片是否自动清空之前编辑对象
    selectDelay number <optional>
    		 2

    选择延迟，距离上次选择多少秒时间内不能触发选择
    capture string <optional>

    选择方式 默认null ，比如：camera 照相机；camcorder 摄像机；microphone 录音。
    Requires:
    • module:libs/shrek/imgslterV.js
    • module:libs/touch/touch.min.js
    Example
    require('ds/createjs/SelectUpImager');
 var _selectUpImager=new ds.createjs.SelectUpImager(250,250,_box);
 _SelectUpImager.on('update',function (e) {
     //选择了一张图片上来时候触发
 });
